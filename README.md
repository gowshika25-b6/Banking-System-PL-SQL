# Banking-System-PL-SQL
Banking System using PL/SQL with procedures, cursors, exception handling and transaction control.

## Project Description
This project implements a banking system using PL/SQL to manage account operations like deposit and withdrawal.

## Features
- Managed account details and balance
- Implemented deposit and withdrawal logic
- Used cursors to display data
- Handled exceptions for invalid accounts

## PL/SQL Concepts Used
- IF-ELSE conditions
- Cursors
- Exception handling
- Transactions

## Learning Outcome
- Real-time problem solving using PL/SQL
- Strong understanding of conditions and logic
- Database transaction handling
